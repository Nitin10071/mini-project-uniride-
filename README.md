# UniRide - Student Ride Sharing App 🚕

UniRide is a mini project web application inspired by Ola/Uber, specially designed for students.
It allows students to offer rides using their personal vehicles and other students can book rides
to reach the university.

This project is built using HTML, CSS, JavaScript, and jQuery.

---

## Features
- Student Login System (Name + Student ID)
- Request Ride System (Passenger Mode)
- Offer Ride System (Driver Mode)
- Ride Matching Algorithm (Pickup-based matching)
- Ride Booking System
- Booking History Storage
- Rating System for Drivers
- Dark/Light Mode Toggle
- LocalStorage Database Simulation

---

## Technologies Used
- HTML
- CSS
- JavaScript
- jQuery
- LocalStorage

---

## Unique Element in this Project ⭐
### Campus-Only GeoFence Ride System
This project contains a unique feature called **Campus-Only GeoFence Mode**.
In this application, rides are strictly limited to one destination:

✅ Chandigarh University

Users cannot choose any other destination. This makes UniRide unique because it is specially
built for a university student environment and prevents misuse for normal city travel.

---

## How to Run
1. Download the project folder.
2. Open `index.html` in any browser.
3. Login with student name and student ID.
4. Offer rides or request rides.
5. Book rides and view booking history.
